ScreenCalc

This project was created to reveal an exact physical screen size despite the screen's aspect. As a terrible virtual keyboard typist, I am concerned about screen width. Accordingly, an iPhone SE is NOT a consideration! A 6.5" screen sounds awesome, unless it has a 20:9 aspect. Its width is not as dramatic as it would be with a 19:9 or 18:9 aspect. This calculator eliminates the size vs. aspect confusion.

See the ScreenCalc web app
