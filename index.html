Screen Size Calculator

This project was created to reveal an exact screen size despite the screen's aspect. I am a terrible virtual keyboard typist and smaller screens only make the problem worse. Screen width determines the density of the keys and I, as I shop for a new phone, am concerned about screen width. Accordingly, an iPhone SE is NOT a consideration! A 6.5" screen sounds awesome, unless it has a 20:9 aspect. It's still a large screen but the width is not as dramatic as would be if it had an 19:9 or 18:9 aspect. I created this calculator to reveal a screen's true physical size. It also includes formatting for various phone aspects and tablets. No screen rotation has been considered.
